## Changes Made
- **Ammo system**: Player now has a limited pool of 10 shots.
- **Enemy Type**: Added a new red spaceships that moves 2x faster. 
- **Fire rate limit**: Added a delay between shots to prevent spamming.  
- **Recoil effect**: Player sprite slightly bounces back when firing.  
- **Dash ability**: Press `Z` to dash. Movement speed temporarily increases to 3× normal speed. Includes a cooldown.  
- **Updated background**: Visual refresh for a clearer game environment.
